A Streamlit app that helps users discover streaming content across platforms based on your individual preferences so you watch a variety of materials from multiple sources instead of being repeatedly served the same time of content by a social platform's algorithm

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://blank-app-template.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
